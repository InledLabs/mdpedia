### React Server Components

Las directivas son necesarias en [React Server Components](https://es.react.dev/reference/rsc/server-components).

Las directivas proporcionan instrucciones para [bundlers compatibles con React Server Components](https://es.react.dev/learn/start-a-new-react-project#bleeding-edge-react-frameworks).
