# Componentes integrados de React – React

React expone algunos componentes integrados que puedes utilizar en tu JSX.

* * *

## Componentes integrados[](#built-in-components "Link for Componentes integrados ")

*   [`<Fragment>`](https://es.react.dev/reference/react/Fragment), escrito alternativamente como `<>...</>`, te permite agrupar varios nodos JSX juntos.
*   [`<Profiler>`](https://es.react.dev/reference/react/Profiler) te permite medir el rendimiento de la renderización de un árbol de React de manera programática.
*   [`<Suspense>`](https://es.react.dev/reference/react/Suspense) te permite mostrar un sustituto mientras los componentes hijos se están cargando.
*   [`<StrictMode>`](https://es.react.dev/reference/react/StrictMode) permite controles adicionales solo para desarrollo que te ayudan a encontrar errores anticipadamente.

* * *
